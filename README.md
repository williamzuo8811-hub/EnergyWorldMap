# 全球能源项目世界地图 🌐

一个**可交互的能源项目世界地图**：在真实可缩放的世界地图上展示全球各国近年的重点能源项目，**放大可逐级看到国家 → 省 → 市 → 街道**（带清晰国界与地名），支持按品类、区域、状态、年份多维筛选，并提供实时统计、项目详情与"最新"动态高亮。

## ✨ 功能

- **真实瓦片地图**：基于 Leaflet，国界、国名、省、市、街道随缩放自然显示。可在右上角切换底图：
  - **暗色科技**（默认，Esri 深色，全球覆盖、加载轻快，与大屏风格统一）
  - **OSM**（OpenStreetMap，全球，街道细节更全）

  > 注：原「高德·街道」「卫星影像」底图为高德 GCJ-02 瓦片，国内服务加载较重、缩放卡顿，已移除；现两个底图均为 WGS-84，坐标对齐更准、无需纠偏。
- **10 大品类**：可再生能源发电、**☢️ 核电**、输变电·电网、储能、工商业·配网大客户、数据中心、大交通（轨交·机场·港口）、石油化工·LNG、矿业，外加 **🤝 国际大客户**（54 家中国出海龙头企业的海外项目，按客户公司归类）。圆点大小≈投资规模，颜色=品类；**各国旗舰项目带脉冲**（每国按投资额取前 2 个，避免满屏脉冲）；输变电走廊/高铁/管道用**流动飞线**表现。
- **密集区聚合（cluster）**：300+ 项目在世界视角自动聚合为带计数的气泡（按主导品类着色），放大逐级散开、点击气泡快速放大；输变电/海缆/高铁**飞线始终常显**，不参与聚合。
- **🔥 投资热力图（宏观格局）**：右上角一键切换「投资热力」模式——按**投资额加权**（√inv 压缩量级，避免巨型项目独占）的密度热力层，蓝→青→绿→黄→红显示**全球能源资本/产能在哪里连成带、哪里最密**（如沙特东部省、红海 NEOM、刚果金铜钴带、中国"沙戈荒"、巴西东北海上风电带）。热力**实时跟随筛选**，并附**分面焦点条**——一键只看某一品类的地理热力（如"全球数据中心热力""全球锂矿热力"）；开启时隐藏标记、保留飞线，右下角附图例。基于本地化的 `Leaflet.heat`。
- **🆕 最新动态**：最近数月（≥ `META.recentSince`，当前 2026-01）有新进展的项目自动标记 🆕，可一键"仅看最新动态"。整库为 2024–2036 前瞻数据集、更新较密，故窗口收窄以保留区分度。
- **10 大品类 × 子分类**：每个品类下再细分子分类（如可再生＝光伏/光热/陆上风电/海上风电/水电/地热/绿氢绿氨；大交通＝高铁/城轨/机场/港口/跨海桥隧；**国际大客户＝按 54 家出海企业归类**——首批 14 家（中国电建/中国能建/国网国际/中石油工程/三峡国际/国家电投/紫金矿业/洛阳钼业/中国有色/比亚迪/宁德时代/格林美/万国DayOne/秦淮数据）＋《特锐德 50 家出海大客户》新增 40 家（中国交建/中铁建/中冶/特变电工/阳光电源/远景/华友/青山/海油工程/中国化学…）），左侧品类项点 ▸ 展开后可按子分类单独开关筛选并看实时计数；详情卡与悬浮提示也会显示"品类 · 子分类"。子分类由 `app.js` 的 `SUB_DEFS` 按关键词/业主自动归类，新增项目无需手工打标签。
- **多维筛选**：品类（含子分类）开关、所属大区、项目状态（规划/在建/投运）、**年份区间双手柄滑块**（覆盖 2024→2036，含"全部 / 🆕最新 / 未来管线 2027+"快捷预设，可单独圈出未来 5 年规划管线）、关键词搜索（项目/国家/业主）。
- **实时统计联动**：顶部 KPI（项目数 / 覆盖国家 / 总投资 / 🆕最新）、右侧分类条形图、区域分布、重点项目 TOP（可切**按投资额 / 按装机容量**排序）。**点右侧分类条 / 区域格即筛选**（与左栏开关双向联动）；地图左上角附**品类色图例**（点击即可切换显示该品类）。
- **📊 硬指标（解析自容量）**：自动把各项目的 `cap` 文本解析成结构化数值，右侧"硬指标"卡按当前筛选实时汇总**装机(GW) / 储能(GWh) / 🧠 智算(EFLOPS) / 线路(km) / 油气产能(万桶/日) / 产能(万吨/年)**——一眼看"在建/规划多少 GW 光伏、多少 GWh 储能、多少 EFLOPS 智算、多少万桶/日炼能"。
- **⚖️ 加权口径 + 装机热力**：地图右上角一键把圆点大小与热力图在**投资额 ↔ 装机容量(MW)** 间切换——即"投资热力"与"装机热力"两套宏观格局。
- **口径修正**：总投资默认**不计入"国际大客户"**（避免与能源品类对同一物理项目重复计，约 5.3%）；投资额统一以**美元**口径展示（详情卡保留原币种），避免币种混杂。
- **▶ 时间轴播放**：一键从 2024 起逐年累计揭示——项目随年份在地图上"长"出来，当年新增项目弹入高亮、地图浮现年份大字，配合未来管线一路演进到 2036，宏观叙事一目了然（拖动滑块/选预设即暂停）。
- **🔗 可分享视图 / ⤓ 导出 / 📸 快照**：一键把当前筛选编码进链接复制分享（打开即还原同一视图，含语言/飞线/热力分面/对比/单项目**深链接**——详情卡 🔗 复制某项目链接，打开即弹该项目并定位）；导出当前筛选结果为 CSV（Excel 中文不乱码）；**📸 一键把当前视图导成信息图快照（PNG，失败回退 SVG）**——含 KPI、分品类、TOP 项目与筛选说明，适合插入文档/PPT 或转发（地图瓦片跨域无法截图，故以统计信息图呈现）。
- **♿ 无障碍 / 键盘可达**：筛选 chips、图例、区域格、企业行等自定义控件可 Tab 聚焦、Enter/Space 触发，有清晰焦点环；详情卡/国别/对比模态支持 Esc 关闭与焦点管理；尊重系统「减少动态效果」偏好（自动关停脉冲/飞线/旋转等动画）。
- **🌐 中英双语**：界面与分析标签中英切换；**全部约 140 个国名**在英文模式下显示英文（悬浮提示/列表/详情卡/国别面板/对比统一，复合国名如"英国—德国"逐段翻译）；项目正文支持可选 `descEn`/`detailEn`，英文模式优先展示英文介绍（已为墨脱、白鹤滩、三峡、NEOM 绿氢、Dogger Bank、Khavda 等旗舰项目补译，其余优雅回退中文并加注）。
- **项目详情卡**：点击地图圆点或右侧列表，查看**一段详细的项目介绍**及容量、投资、业主、状态、最近动态时间等。
- **🌍 国别下钻仪表盘**：在详情卡点击"国家/地区 🔎"，弹出该国能源组合面板——项目数 / 总投资 / 总装机 / 🆕近一年 KPI、分品类（项目数·投资额）、状态分布、里程碑年份分布、重点项目 TOP（可再点进单个项目），把"沙特/巴西/澳洲未来 5 年"这类深挖变成可点击的国别仪表盘。
- **🆚 国别对比**：地图右上"🆚 对比"按钮（或国别面板的"⊕ 加入对比"），把最多 4 个国家的能源组合**并排小多图对比**——各自 KPI、分品类投资条、TOP3 项目，配可搜索的国家选择器，一眼看清"沙特 vs 巴西 vs 印度"谁押注哪个赛道。
- **🎯 国际大客户 BD 看板**：地图右上"🎯 客户"按钮，把 54 家中国出海大客户按 **BD 梯队（第一/二/三）× 产品关联度（极高/高/中高）** 分组成一张"目标客户作战图"——每家显示客户类型、特锐德重点关联产品、典型海外场景、海外项目数与覆盖国家数，**点公司即在地图上筛出其全部海外项目并飞到其范围**；元数据见 `js/clients-meta.js`（源自《特锐德 50 家出海大客户》清单）。
- **📍 最新进展**：每个项目都附一条简洁的最新进展（如"2025-09 全容量并网"/"2026 年推进 FID"），在详情卡顶部高亮显示、悬浮提示也可见，一眼掌握项目当前到了哪一步。
- **📱 移动端适配**：窄屏下筛选栏 / 统计栏收为可滑出的抽屉（🔍 / 📊 浮动按钮调出），手机上也能完整筛选、搜索、看统计。
- **纳入门槛**：投资额 > 5000 万元人民币（约 700 万美元）的项目均可收录。

## 🚀 打开方式

**方式一（最简单）**：直接双击 `index.html` 用浏览器打开。地图库与数据已本地化，**底图瓦片需联网加载**（省/市级地名细节本身依赖在线地图服务）。

**方式二（本地服务器，可选）**：

```powershell
powershell -ExecutionPolicy Bypass -File serve.ps1 -Port 4173
```

然后浏览器打开 `http://localhost:4173/`。推荐 Chrome / Edge。

## ✏️ 如何新增 / 修改项目（满足"有新项目就补充、有变化就更新"）

打开 `js/data.js`，在 `PROJECTS` 数组里增删改条目即可。一个项目长这样：

```js
{
  id: 100,                         // 唯一编号
  name: '某海上风电场',            // 中文名称
  en: 'XXX Offshore Wind',         // 英文/原文名（可空）
  country: '中国',                 // 国家/地区
  region: '中国',                  // 大区：中国/东亚/东南亚/南亚/中亚/中东/欧洲/北美/南美/非洲/大洋洲
  cat: 'renewable',                // 品类键（见下表）
  coord: [121.5, 31.0],            // [经度, 纬度]（WGS-84，高德底图会自动 GCJ-02 纠偏）
  cap: '1.2 GW',                   // 容量/规模
  inv: 30,                         // 投资额（单位：亿美元，用于统计）
  invText: '约 30 亿美元',         // 投资额展示文本
  status: '在建',                  // 规划 / 在建 / 投运  ← 动态变化时改这里
  year: 2026,                      // 关键里程碑年份
  updated: '2026-06',              // 最近动态时间；≥ META.recentSince 自动标 🆕
  owner: 'XX 集团',                // 业主/参与方
  flagship: true,                  // 是否旗舰（地图上加脉冲高亮）
  desc: '一句话摘要。',            // 悬浮提示/列表用的短摘要
  detail: '一段详细的项目介绍，背景+规模+意义+最新进展。', // ★ 点击项目后详情卡展示
  route: [[121.5,31.0],[122.0,30.5]] // 可选：连线（输变电/高铁/管道），两点及以上
}
```

- **新项目**：复制一条、改字段、把 `updated` 写成当月，即自动出现并标 🆕。
- **现有项目变化**：改 `status` / `cap` / `inv` / `desc` / `detail`，并更新 `updated`。
- **"最近一年"阈值**：在 `data.js` 顶部 `META.recentSince`（当前 `'2025-06'`）与 `META.lastUpdated`（`'2026-06'`，顶栏显示）。

**品类键对照表**（`cat` 字段取值）：

| 键 | 品类 | 颜色 | 键 | 品类 | 颜色 |
|---|---|---|---|---|---|
| `renewable` | 可再生能源发电 | 绿 | `datacenter` | 数据中心 | 粉 |
| `nuclear` | 核电 | 黄 | `transport` | 大交通 | 蓝 |
| `grid` | 输变电·电网 | 青 | `petro` | 石油化工·LNG | 红 |
| `storage` | 储能 | 紫 | `mining` | 矿业 | 金 |
| `ci` | 工商业·配网 | 橙 | `client` | 国际大客户 | 黄绿 |

> 注：原"通信·海缆"类别已按需求移除。

> 想新增品类？在 `data.js` 顶部 `CATEGORIES` 加一项（`name/short/color/icon`），地图、图例、统计会自动适配。
